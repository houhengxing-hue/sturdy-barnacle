# 中国乘用车市场日报

每日收集中国乘用车市场信息的展示网站，包括：

- **产业政策** - 国家及部委级汽车产业政策
- **地方补贴政策** - 各省市购车补贴、置换补贴等
- **品牌营促销活动** - 主流品牌新车/二手车促销
- **新产品上市** - 新车发布、改款、预售等
- **社交媒体口碑** - 微博、小红书、懂车帝等平台舆情

## 技术栈

- React 18
- Vite 5
- Tailwind CSS 3

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
